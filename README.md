Snake Game

Ein kleines Java Snake Game mit GUI, Login-System und Highscore-Saving.

Das Projekt wurde mit Java Swing gemacht und ist gedacht zum Lernen von:

GUI Programming
OOP in Java
Event Handling
Timer & Threads
Datei speichern/lesen (txt Files)

Features:

- Login Screen
- Snake Game mit GUI
- Highscores werden in einer .txt Datei gespeichert
- Restart Funktion
- Deutsche Oberfläche
- Keyboard Controls
- Clean & simple Code Structure

Projekt starten:

Projekt in IntelliJ IDEA öffnen
Main.java starten
Username eingeben
Enjoy the game!


Controls:

Taste	Funktion
↑ ↓ ← →	Snake bewegen
R	Restart
H	Highscores anzeigen
